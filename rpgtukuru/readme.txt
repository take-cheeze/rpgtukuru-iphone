== 主なソースの説明 ==
=== ベース ===
* game.cpp/h
  ゲームの大元。タイトルやフィールドなどの切り替え管理

* game_field.cpp/h
  フィールドマップの処理

* game_map.cpp/h
  フィールドマップ描画

* game_event_manager.cpp/h
  イベント処理マネージャ

* game_chara.cpp/h、game_player.cpp/h、game_npc.cpp/h
  フィールドのキャラ制御

* game_chara_status.cpp/h
  プレイヤーのステータス

* game_inventory.cpp/h
  所持品やお金管理

* game_system.cpp/h
  必要なものを色々持ってるクラス。いろんなところからここを通してアクセス。

* game_battle〜
  バトル関係

* game〜menu
  メニュー関係
